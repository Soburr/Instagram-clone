# Instagram Clone

A modern social media web application inspired by Instagram, built with Laravel.  
The platform allows users to create profiles, share posts, interact with content in real time, and manage social interactions through a clean and scalable architecture.

---

## 🚀 Features

### Authentication & User Management
- User registration and login
- Secure authentication system
- Profile management
- Password reset functionality

### Social Features
- Create, edit, and delete posts
- Upload images and media
- Like and unlike posts
- Comment system
- Follow and unfollow users
- Personalized feed system

### Real-Time Functionality
- Real-time notifications
- Live updates for interactions
- Dynamic content rendering

### Media Handling
- Cloud image storage
- Optimized image uploads
- Responsive media display

### Admin & Moderation
- User management
- Content moderation
- Post reporting system

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Soburr/your-repo-name.git
```

Navigate into the project directory:
```bash   
cd your-repo-name
```
Install backend dependencies:
```bash
composer install
```

Install frontend dependencies:
```bash
npm install
```

Create environment file:
```bash
cp .env.example .env
```

Generate application key:
```bash
php artisan key:generate
```

Run migrations:
```bash
php artisan migrate
```

Start development server:
```bash
php artisan serve
```

Run frontend build:
```bash
npm run dev
